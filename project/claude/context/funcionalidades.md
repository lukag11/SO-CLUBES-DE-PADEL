# Funcionalidades

**Actualizado:** 2026-05-04

---

## MVP — Implementado ✅

- Registro y login de jugadores
- Reservas de canchas (grilla real, slots 1.5h)
- Turnos fijos semanales con aprobación
- Gestión básica de usuarios (jugadores, admin, profesor)
- Notificaciones (admin + jugador)
- Pagos (registro manual)
- Configuración del club (canchas, horarios, logo, landing)

## V1 — Implementado ✅

- Torneos automáticos (Round Robin, Eliminación directa, Grupos + Eliminación)
- Ranking de jugadores (estadísticas básicas)
- Historial de partidos
- Dashboard del profesor

## V2 — Pendiente 🔲

- Pagos online (integración Mercado Pago / Stripe)
- Estadísticas avanzadas por jugador y club
- App mobile (React Native)
- Backend real con API REST

## V3 — Futuro 🔲

- IA para armar partidos equilibrados (matchmaking por nivel)
- WhatsApp chatbot para reservas y notificaciones
- Análisis de rendimiento avanzado
- Subdominios por club (lospinos.padelos.com)
- Panel SuperAdmin para gestión de todos los tenants
- Registro self-service de clubes con pago online
