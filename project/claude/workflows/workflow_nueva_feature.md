# 🚀 Workflow: Nueva Feature

## 🎯 Objetivo

Definir correctamente una nueva funcionalidad antes de implementarla.

---

## 🧩 Pasos

1. Usar el master prompt
2. Describir la idea de la feature
3. Usar plantilla_feature.md
4. Completar:
   - objetivo
   - reglas de negocio
   - edge cases

---

## ⚠️ Importante

- NO escribir código en esta etapa
- Validar lógica antes de implementar

---

## ✅ Resultado esperado

- Feature bien definida
- Lista para dividir en tareas
