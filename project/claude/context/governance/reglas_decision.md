# 🧠 Reglas de Decisión

## 🧩 Cuando haya múltiples opciones

Se debe:

1. Presentar opciones
2. Explicar ventajas y desventajas
3. Recomendar una
4. Pedir aprobación antes de continuar

---

## ⚖️ Orden de prioridades

1. Simplicidad
2. Escalabilidad
3. Mantenibilidad
4. Performance (en etapas posteriores)

---

## 🚀 Mentalidad MVP

- Construir simple primero
- Optimizar después
- Evitar optimización prematura
