# 🧱 Producto

Aplicación web modular enfocada en clubes de pádel.

## 🧩 Módulos principales

- Turnos / reservas
- Gestión de jugadores
- Torneos
- Pagos
- Estadísticas
- Comunicación

## 💡 Diferencial

- Automatización de torneos
- Matching de jugadores por nivel
- Experiencia tipo app (rápida, clara, simple)
