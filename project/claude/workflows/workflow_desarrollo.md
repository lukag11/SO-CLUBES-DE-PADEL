# ⚙️ Workflow: Desarrollo

## 🎯 Objetivo

Implementar una funcionalidad de forma controlada y escalable.

---

## 🧩 Pasos

1. Tomar una feature definida
2. Dividir en tareas (plantilla_tarea.md)
3. Usar prompt de iniciar tarea
4. Implementar paso a paso
5. Validar cada cambio

---

## ⚠️ Reglas

- Cambios pequeños
- No modificar múltiples cosas a la vez
- Siempre explicar antes de implementar

---

## ✅ Resultado esperado

- Código funcional
- Código limpio
- Sin romper otras partes
